# Research Methodology

When conducting research, follow these steps:

1. **Define scope** — Identify the key questions and boundaries of the investigation
2. **Gather information** — Use `http_request` to fetch data from multiple authoritative sources
3. **Cross-reference** — Validate findings across sources, note contradictions
4. **Analyze** — Synthesize findings into actionable insights
5. **Structure output** — Produce a technical brief with:
   - Executive summary (2-3 sentences)
   - Key findings (bulleted)
   - Analysis (detailed sections)
   - Recommendations
   - Sources and confidence levels

Always cite sources and indicate confidence level (high/medium/low) for each finding.
